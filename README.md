# ZMK Config — Lily58 Pro

Персональная конфигурация прошивки [ZMK](https://zmk.dev/) для сплит-клавиатуры **Lily58 Pro** (беспроводная, BLE).

## Клавиатура

- **Модель**: Lily58 Pro
- **Микроконтроллер**: nice!nano v2
- **Подключение**: Bluetooth Low Energy (BLE), беспроводная сплит-конфигурация
- **Прошивка**: ZMK v0.3

---

## Раскладка

Клавиатура использует **4 слоя**, активируемых большими пальцами.

### Layer 0 — Base (QWERTY)

├─────┼───┼───┼───┼───┼───┤               ├───┼───┼───┼───┼───┼─────┤
│ TAB │ Q │ W │ E │ R │ T │               │ Y │ U │ I │ O │ P │  -  │
├─────┼───┼───┼───┼───┼───┤               ├───┼───┼───┼───┼───┼─────┤
│CTRL │ A │ S │ D │ F │ G │               │ H │ J │ K │ L │ ; │  '  │
├─────┼───┼───┼───┼───┼───┼────┐     ┌────┼───┼───┼───┼───┼───┼─────┤

              │ALT│GUI│LWR│SPC │     │ENT │RSE│BSP│PSCR│
              └───┴───┴───┴────┘     └────┴───┴───┴────┘

### Layer 1 — Lower (удержание LWR)

F-клавиши и горячие клавиши редактирования — все на левой стороне.

```
┌────┬────┬────┬────┬────┬────┐               ┌────┬────┬────┬────┬────┬────┐
│ F1 │ F2 │ F3 │ F4 │ F5 │ F6 │               │    │    │    │    │    │    │
├────┼────┼────┼────┼────┼────┤               ├────┼────┼────┼────┼────┼────┤
│ F7 │ F8 │ F9 │F10 │F11 │F12 │               │    │    │    │    │    │    │
├────┼────┼────┼────┼────┼────┤               ├────┼────┼────┼────┼────┼────┤
│    │C+A │    │    │    │    │               │    │    │    │    │    │    │
├────┼────┼────┼────┼────┼────┼────┐     ┌────┼────┼────┼────┼────┼────┼────┤
│    │C+Z │C+X │C+C │C+V │    │    │     │    │    │    │    │    │    │    │
└────┴────┴────┼────┼────┼────┼────┘     └────┼────┼────┼────┴────┴────┴────┘
              │    │    │[LWR]│    │     │    │[RSE]│    │    │
              └────┴────┴─────┴────┘     └────┴─────┴────┴────┘
```

| Клавиша | Действие |
|---------|----------|
| `C+A` | Выделить всё (Ctrl+A) |
| `C+Z` | Отменить (Ctrl+Z) |
| `C+X` | Вырезать (Ctrl+X) |
| `C+C` | Копировать (Ctrl+C) |
| `C+V` | Вставить (Ctrl+V) |

### Layer 2 — Raise (удержание RSE)

Навигация, управление окнами, стрелки и математические операторы.

```
┌────┬────────┬────────┬────────┬────┬────┐               ┌──────┬──────┬──────┬──────┬──────┬────┐
│    │        │        │        │    │    │               │PSCR  │ DEL  │      │      │      │    │
├────┼────────┼────────┼────────┼────┼────┤               ├──────┼──────┼──────┼──────┼──────┼────┤
│    │C+Sh+Tab│ Win+↑  │ C+Tab  │    │    │               │      │  ↑   │      │      │      │    │
├────┼────────┼────────┼────────┼────┼────┤               ├──────┼──────┼──────┼──────┼──────┼────┤
│    │C+Win+← │ Win+↓  │C+Win+→ │    │    │               │  ←   │  ↓   │  →   │      │      │    │
├────┼────────┼────────┼────────┼────┼────┼────┐     ┌────┼──────┼──────┼──────┼──────┼──────┼────┤
│    │        │        │        │    │    │    │     │    │  +   │  -   │  =   │  *   │  /   │    │
└────┴────────┴────────┼────────┼────┼────┼────┘     └────┼──────┼──────┼──────┴──────┴──────┴────┘
                       │        │    │[LWR]│    │   │    │[RSE] │      │      │
                       └────────┴────┴─────┴────┘   └────┴──────┴──────┴──────┘
```

| Комбо | Действие |
|-------|----------|
| `C+Sh+Tab` | Предыдущая вкладка браузера |
| `C+Tab` | Следующая вкладка браузера |
| `Win+↑` / `Win+↓` | Snap окна вверх/вниз |
| `C+Win+←` / `C+Win+→` | Переключение виртуальных рабочих столов |

### Layer 3 — Adjust (Lower + Raise одновременно)

Bluetooth, питание, ZMK Studio.

```
┌──────┬──────┬──────┬──────┬──────┬──────┐               ┌────────┬────┬────┬────┬────┬────────┐
│BTCLR │ BT0  │ BT1  │ BT2  │ BT3  │ BT4  │               │EP_TOG  │    │    │    │    │STUDIO  │
├──────┼──────┼──────┼──────┼──────┼──────┤               ├────────┼────┼────┼────┼────┼────────┤
│      │      │      │      │      │      │               │        │    │    │    │    │ RESET  │
├──────┼──────┼──────┼──────┼──────┼──────┤               ├────────┼────┼────┼────┼────┼────────┤
│      │      │      │      │      │      │               │        │    │    │    │    │        │
├──────┼──────┼──────┼──────┼──────┼──────┼────┐     ┌────┼────────┼────┼────┼────┼────┼────────┤
│      │      │      │      │      │      │    │     │    │        │    │    │    │    │        │
└──────┴──────┴──────┼──────┼──────┼──────┼────┘     └────┼────────┼────┼────┴────┴────┴────────┘
                     │      │      │[LWR] │    │     │    │[RSE]   │    │    │
                     └──────┴──────┴──────┴────┘     └────┴────────┴────┴────┘
```

| Клавиша | Действие |
|---------|----------|
| `BTCLR` | Очистить текущий BT профиль |
| `BT0–BT4` | Переключиться на BT устройство 0–4 |
| `EP_TOG` | Переключить внешнее питание |
| `STUDIO` | Разблокировать ZMK Studio |
| `RESET` | Перезагрузить правую половину |

## Конфигурация

### Режим сна

| Параметр | Значение | Описание |
|----------|----------|----------|
| `CONFIG_ZMK_IDLE_TIMEOUT` | 15 мин | Таймер бездействия (левая) |
| `CONFIG_ZMK_IDLE_SLEEP_TIMEOUT` | 30 мин | Глубокий сон (обе половины) |

Каждая половина засыпает **независимо** по своему таймеру бездействия.

### OLED

- OLED 128x32 включён только на левой половине
- Для левой сборки подключён дополнительный shield `nice_oled`
- На экране слева отображаются активный слой вертикальным стеком и заряд левой половины
- Настройка экрана задаётся через `config/lily58_left.conf`

### Bluetooth

- Усиленная мощность передатчика: `CONFIG_BT_CTLR_TX_PWR_PLUS_8=y`
- Уровень заряда правой половины отображается на левой: `CONFIG_ZMK_SPLIT_BLE_CENTRAL_BATTERY_LEVEL_FETCHING=y`

### ZMK Studio

Включён (`CONFIG_ZMK_STUDIO=y`) — позволяет редактировать раскладку без перепрошивки через [ZMK Studio](https://zmk.studio/).

## Прошивка

Прошивка собирается автоматически через **GitHub Actions** при каждом пуше.

1. Запушить изменения в `main`
2. Перейти в **Actions** → последний успешный workflow
3. Скачать артефакт `firmware.zip`
4. Прошить каждую половину:
   - Зажать кнопку Reset дважды (двойной клик) → появится USB-накопитель `NICENANO`
   - Скопировать `lily58_left-nice_nano_v2-zmk.uf2` на левую половину
   - Скопировать `lily58_right-nice_nano_v2-zmk.uf2` на правую половину
